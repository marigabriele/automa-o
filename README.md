# automação
Automação em Python para cadastrar produtos automaticamente em um sistema, evitando cadastro manual e economizando tempo em tarefas repetitivas dentro de empresas.
Este projeto é uma automação desenvolvida em Python para cadastrar produtos automaticamente em um sistema ou site.
# objetivo
Em muitas empresas, o cadastro manual de produtos (código, preço, tipo, descrição, etc.) pode levar horas e ser repetitivo. Para resolver esse problema, desenvolvi uma automação que lê os dados de uma planilha Excel e realiza o cadastro automaticamente.

A automação utiliza bibliotecas para manipulação de dados e controle do computador, tornando o processo mais rápido, eficiente e com menos erros.

# Tecnologias utilizadas:
- Python
- pandas
- openpyxl
- pyautogui
